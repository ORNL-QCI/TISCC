Detailed Description

Represents a named instruction acting on the qubit labels (a, b, c, d, or m) of a general surface code plaquette.

Instructions are primarily constructed within LogicalQubit::init_circuits as members of syndrome extraction circuits. These are intended to be QASM-like instructions that act on qubit labels rather than qsites. The Instruction constructor is private. There are no guard-rails as to acceptable instruction names except that there should be a corresponding member function within HardwareModel to compile it into HW_Instructions, typically through LogicalQubit::apply_instruction.
