Detailed Description

Represents a named instruction acting at a particular qsite(s) on the hardware grid at a specific time.

Instructions are compiled into HW_Instructions by LogicalQubit methods using the member functions of HardwareModel. The names are intended to be members of a hardware gate set defined within HardwareModel::init_TI_ops. HW_Instructions act on qsites rather than qubits.
